\doxysubsection{Detailed Description}
A container managing a map data structure that maps \mbox{\hyperlink{classParameter}{Parameter}} combinations to their respective found values. 

The class manages concurrent access using the \mbox{\hyperlink{classValueMap_adfb0c4d55f17d85cd53186d0be2c1cfb}{operations\+Lock}}. Running median calculation is supported by using sets \mbox{\hyperlink{classValueMap_add6b191069b0975d94d5bdb695992936}{upper\+Values}} and \mbox{\hyperlink{classValueMap_ac4dcc6213ff75c29ca81e1bd586f7b73}{lower\+Values}}. Values are inserted into the data structure at once when \mbox{\hyperlink{classValueMap_aa13ab1a81147776fe1b76c1342b2dbd5}{update\+Map}} is called. Before that they are saved in \mbox{\hyperlink{classValueMap_ab3d7fb7bf10aa25f98c3555e972ebcbd}{tba}} to avoid unnecessary costly insertion operations.
