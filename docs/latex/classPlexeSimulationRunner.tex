Creates \mbox{\hyperlink{classPlexeSimulationRunner}{Plexe\+Simulation\+Runner}} which cannot use more than the given number of threads. 

Number of repetitions, scenarios to be simulated and the \mbox{\hyperlink{classConfigEditor}{Config\+Editor}} must also be defined. The new \mbox{\hyperlink{classPlexeSimulationRunner}{Plexe\+Simulation\+Runner}} uses $ t = \min(\mathit{threads}, \mathit{repeat} \cdot \mathit{size}(\mathit{scenarios})) $ concurrent threads for parallelization of \mbox{\hyperlink{classPlexeSimulationRunner_a1e79cf0374bf9280cd0293fb037c531e}{work(std\+::pair$<$ std\+::filesystem\+::path, std\+::pair$<$ std\+::basic\+\_\+string$<$ char $>$, unsigned int $>$$>$)}}. For the parallelization of \mbox{\hyperlink{classPlexeSimulationRunner_a323b762ef210ebdb846cd3456bd20f67}{work(vector$<$shared\+\_\+ptr$<$\+Parameter$>$$>$)}} $ t\textnormal{\textquotesingle} = \lfloor \mathit{threads} \div t \rfloor $ concurrent threads are used.
