Calculates the lower right convex hull of a set of points. 

Points are defined by the given Hy\+Rects diagonal length (x axis) and average value (y axis). For each returned \mbox{\hyperlink{classHyRect}{Hy\+Rect}} the slope to the point right of it is returned (if it is the rightmost point, infinity is chosen). That slope value can be used by DIRECT as the highest Lipschitz constant for which the \mbox{\hyperlink{classHyRect}{Hy\+Rect}} satisfies the first condition.
