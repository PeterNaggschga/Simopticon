\doxysubsection{Detailed Description}
This module extends \mbox{\hyperlink{classOptimizer}{Optimizer}} to use a variant of the DIRECT algorithm by \href{https://doi.org/10.1007/BF00941892}{\texttt{ Jones et al}}. 

It incorporates features proposed by \href{https://doi.org/10.1007/s10898-016-0447-z}{\texttt{ Liu et al.}} and \href{https://doi.org/10.1137/040621132}{\texttt{ Sergeyev and Kvasov}}.
