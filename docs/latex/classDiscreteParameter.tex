\doxysubsection{Detailed Description}
Implements a \mbox{\hyperlink{classParameter}{Parameter}} using discrete values. 

The value of the \mbox{\hyperlink{classParameter}{Parameter}} is calculated as $ \mathit{val} = \mathit{times} \cdot \mathit{step} + \mathit{offset} $.
