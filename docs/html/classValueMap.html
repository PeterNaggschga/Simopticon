Detailed Description

A container managing a map data structure that maps parameterCombinations to their respective found values.

The class manages concurrent access using the operationsLock. Running median calculation is supported by using sets upperValues and lowerValues. Values are inserted into the data structure at once when updateMap is called. Before that they are saved in tba to avoid unnecessary costly insertion operations.
