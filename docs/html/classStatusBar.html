Updates the output in the command line with gathered information from the used Optimizer, SimulationRunner and Evaluation.

If the current optimum or the step the optimization is in has changed since the last call, the whole output is printed again. Otherwise only the progress of the active component obtained by Status::getStatusBar is updated.

Parameters
optPointer to Optimizer used in optimization.
runnerPointer to SimulationRunner used in optimization.
evalPointer to Evaluation used in optimization.
currentValParameter combination and respective value of the current optimum.
stepChangedBoolean defining whether the current step has changed since the last call.
currentStepCurrent step the optimization is in.
